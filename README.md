## Getting Started âš¡

Prerequisites  
- Python 3.8+  
- Node.js & npm  
- Django & React.js installed  

Installation Steps
1. Clone the repository:  
   git clone https://github.com/Jason3105/CollegePortal.git  
   cd CollegePortal  

2. Backend Setup:  
   pip install -r requirements.txt  
   python manage.py migrate  
   python manage.py runserver

3. Frontend Setup:  
   cd main  
   npm install  
   npm start
